Container

Container tag are the labels that contain a few information such as content, picture, etc. There are a few Container tag in HTML.

1. div tag

Division tag or div tag is used to make blocks or divisions in the document.

2. span tag

This is used for inline content

3. p tag

This is used for paragraph

4. pre tag

This is used for pre formatted text

5. code tag

This is used to represent as source code

Text Formatting

Text formatting is used to format the text/data of html pages. You can bold, emphasis the text to make document attractive.

1. b tag

This is used to bold the text.

2. strong tag

This is also used to bold the text.

3. i tag

This is used to apply italicize the text.

4. em tag

This is also used to apply italicize the text.

5. sub tag

This is used for subscript.

6. sup tag

This is used for superscript.

Characters & Symbols

Some symbols are not directly present on the keyboard, but there are some ways to use them in HTML documents. We can display them either by entity name, decimal, or hexadecimal value.

Semantic Elements

Semantic elements are those elements that are self describable, i.e., from their name itself, you can understand their meaning.
